Frequently Asked Questions

Find answers to common questions about KyOS installation, configuration, and usage.

General Questions

What is KyOS?

KyOS is a modern, powerful, and beautiful Arch Linux-based distribution featuring the Qtile window manager, a user-friendly TUI installer, and carefully curated desktop environment designed for developers and enthusiasts.

Is KyOS suitable for beginners?

Yes! While KyOS is based on Arch Linux, it includes an intuitive installer and pre-configured desktop environment that makes it accessible to users new to Linux. The TUI installer handles most of the complex setup automatically.

What desktop environment does KyOS use?

KyOS uses Qtile, a highly configurable tiling window manager written in Python. It comes pre-configured with beautiful themes, useful widgets, and sensible keybindings while remaining fully customizable.

How often is KyOS updated?

KyOS follows a rolling release model based on Arch Linux, meaning you receive the latest updates continuously. Major ISO releases are created periodically with updated installer features and system improvements.

Installation Questions

What are the minimum system requirements?

KyOS requires:

  • 64-bit processor (x86_64)
  • 2 GB RAM minimum (4 GB recommended)
  • 20 GB free disk space
  • Internet connection for installation
  • UEFI or Legacy BIOS support

Can I dual boot KyOS with Windows?

Yes! The KyOS installer includes dual-boot support. You can install KyOS alongside Windows by selecting the "Dual Boot (Unallocated Space)" option in the installer. Make sure to have unallocated space available on your drive.

Do I need to manually partition my drive?

No, manual partitioning is optional. The installer offers three options:

  • Full Disk Installation: Automatic partitioning (easiest)
  • Dual Boot: Install alongside existing OS
  • Manual Partitioning: Full control over partitions (advanced)

What if the installation fails?

If installation fails:

  1. Check the error message displayed
  2. Ensure your system meets minimum requirements
  3. Verify internet connectivity
  4. Try booting with different kernel parameters
  5. Check our troubleshooting guide

Can I install KyOS in a virtual machine?

Yes! KyOS works well in virtual machines. The installer includes VM-specific optimizations. Recommended VM settings:

  • 4 GB RAM or more
  • 30+ GB disk space
  • Enable 3D acceleration if available
  • 2+ CPU cores for better performance

Usage Questions

How do I install software on KyOS?

KyOS uses pacman (Arch Linux package manager) and supports AUR packages:

# Install from official repositories
doas pacman -S package-name

# Search for packages
pacman -Ss search-term

# Install AUR packages (install yay first)
yay -S aur-package-name

Why does KyOS use 'doas' instead of 'sudo'?

KyOS uses doas for security reasons. Doas is simpler, more secure, and has a smaller attack surface than sudo. It provides the same functionality with better security defaults. If you prefer sudo, you can install it: doas pacman -S sudo

How do I update my system?

Update your KyOS system regularly:

# Update package database and all packages
doas pacman -Syu

# Force refresh package database
doas pacman -Syyu

How do I change the desktop theme?

KyOS themes can be customized by editing configuration files:

  • Qtile config: ~/.config/qtile/config.py
  • Terminal colors: ~/.config/alacritty/alacritty.yml
  • GTK themes: ~/.config/gtk-3.0/settings.ini

See our customization guide for detailed instructions.

What are the essential Qtile keybindings?

Here are the most important keybindings in KyOS:

  • Super + Enter: Open terminal
  • Super + r: Application launcher
  • Super + q: Close window
  • Super + Tab: Switch between windows
  • Super + [1-9]: Switch to workspace
  • Super + Shift + [1-9]: Move window to workspace

See the complete list in our keybindings guide.

Troubleshooting

My WiFi isn't working after installation

Try these solutions:

  1. Check if NetworkManager is running: systemctl status NetworkManager
  2. Start NetworkManager: doas systemctl enable --now NetworkManager
  3. Connect to WiFi: nmcli device wifi connect "SSID" password "password"
  4. Check for driver issues: lspci -k | grep -A3 Network

Audio isn't working

Common audio fixes:

  1. Check if audio is muted: pactl list sinks
  2. Unmute audio: pactl set-sink-mute @DEFAULT_SINK@ false
  3. Install additional packages: doas pacman -S pulseaudio-alsa alsa-utils
  4. Add user to audio group: doas usermod -a -G audio $USER

Display resolution is wrong

Fix display resolution:

# List available resolutions
xrandr

# Set specific resolution
xrandr --output HDMI-1 --mode 1920x1080

# Make permanent by adding to ~/.xinitrc
echo "xrandr --output HDMI-1 --mode 1920x1080" >> ~/.xinitrc

Package installation fails

If package installation fails:

  1. Update package database: doas pacman -Sy
  2. Check internet connection: ping archlinux.org
  3. Clear package cache: doas pacman -Scc
  4. Check for conflicting packages: pacman -Qdt

Advanced Questions

Can I customize the Qtile configuration?

Absolutely! Qtile is highly customizable through Python configuration:

  • Edit ~/.config/qtile/config.py
  • Modify layouts, keys, widgets, and themes
  • Restart Qtile: Super + Ctrl + r
  • Check Qtile basics for more details

How do I enable AUR packages?

Install an AUR helper like yay:

# Clone and install yay
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si

# Now you can install AUR packages
yay -S visual-studio-code-bin
yay -S google-chrome

How do I backup my KyOS configuration?

Backup important configuration files:

# Backup home directory configs
tar -czf kyos-backup.tar.gz ~/.config ~/.bashrc ~/.xinitrc

# Or use git for dotfiles management
cd ~
git init
git add .config .bashrc .xinitrc
git commit -m "Initial dotfiles backup"

Can I contribute to KyOS development?

Yes! KyOS welcomes contributions:

  • Report bugs and issues
  • Submit feature requests
  • Contribute code improvements
  • Help with documentation
  • Share your customizations

Check our contributing guide for more information.
