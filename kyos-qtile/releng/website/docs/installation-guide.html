KyOS Installation Guide

Complete step-by-step guide to installing KyOS on your system. This guide covers everything from preparation to post-installation setup.

Overview

KyOS features an advanced TUI (Text User Interface) installer that makes installation simple and straightforward. The installer supports multiple partitioning modes including dual-boot scenarios and provides automatic configuration for most hardware.

What makes KyOS installer special?

  • Beautiful TUI with intuitive navigation
  • Automatic privilege escalation - no sudo required
  • Smart dual-boot detection and setup
  • Automatic dotfiles and VM configuration
  • Comprehensive error handling and logging

System Requirements

Minimum Requirements

  • Architecture: 64-bit (x86_64) processor
  • Memory: 2 GB RAM (4 GB recommended)
  • Storage: 20 GB free disk space (30 GB recommended)
  • Graphics: Any graphics card with basic 2D acceleration
  • Network: Internet connection for package downloads

Recommended Specifications

  • Memory: 8 GB RAM or more
  • Storage: SSD with 50 GB+ free space
  • Graphics: Dedicated GPU for better performance
  • Network: Ethernet connection for faster downloads

Preparation

1

Download KyOS ISO

Download the latest KyOS ISO from the official website and verify its integrity.

# Download the ISO (example)
wget https://kyos.org/downloads/kyos-2024.01-x86_64.iso

# Verify checksum
sha256sum kyos-2024.01-x86_64.iso
2

Create Bootable USB

Create a bootable USB drive using your preferred method:

# Using dd command (Linux/macOS)
sudo dd if=kyos-2024.01-x86_64.iso of=/dev/sdX bs=4M status=progress

# Using Rufus (Windows) or Etcher (cross-platform) is recommended

Warning

Make sure to replace /dev/sdX with your actual USB device. This operation will erase all data on the USB drive.

3

Backup Important Data

Before installation, ensure you have backed up any important data from your system.

Booting from Live ISO

1

Boot from USB

Insert the USB drive and boot from it. You may need to:

  • Access BIOS/UEFI settings (usually F2, F12, or Delete during boot)
  • Disable Secure Boot if necessary
  • Set USB as first boot device
2

Live Environment

KyOS will boot directly into the Qtile desktop environment without requiring a display manager. You'll be automatically logged in as the 'live' user.

Running the Installer

1

Launch Installer

Open a terminal and run the installer:

# Simply run the installer command
kyosinstall

No sudo required! The installer handles privilege escalation automatically.

2

Choose Installation Mode

Select between:

  • Automated: Uses sensible defaults for quick installation
  • Interactive: Full control over all settings

Partitioning Options

Full Disk Installation

Erases the entire selected disk and installs KyOS. Recommended for dedicated systems.

Dual Boot Installation

Installs KyOS alongside existing operating systems using unallocated space. The installer will automatically detect and preserve existing OS installations.

Dual Boot Requirements

  • Minimum 30 GB unallocated space
  • Existing OS should have its bootloader intact
  • UEFI systems: EFI system partition will be shared

Manual Partitioning

For advanced users who want complete control over partition layout. The installer will launch a partition manager and guide you through the process.

First Boot

1

Automatic Login

After installation, your system will automatically log you into the Qtile desktop environment.

2

Dotfiles Installation

KyOS automatically installs curated dotfiles from the kyos repository, providing a beautiful and functional desktop out of the box.

3

VM Optimizations

If running in a virtual machine, additional configurations are automatically applied for optimal performance.

System Updates

Update System

Keep your KyOS installation up to date:

# Update package database and system
doas pacman -Syu

# Update AUR packages (if enabled)
yay -Syu

Customization

Qtile Configuration

Customize your Qtile window manager:

# Edit Qtile configuration
nano ~/.config/qtile/config.py

# Restart Qtile to apply changes
# Use Mod+Ctrl+R (default keybinding)

Theme Switching

KyOS comes with multiple themes that can be easily switched using the included scripts.
