Custom Keybindings: You can modify or add your own keybindings by editing ~/.config/qtile/config.py. Always backup your config before making changes!

Customizing Keybindings

Learn how to create your own keyboard shortcuts and modify existing ones.

Adding Custom Keybindings

# In ~/.config/qtile/config.py, add to the keys list:

# Application launchers
Key([mod], "b", lazy.spawn("firefox"), desc="Launch Firefox"),
Key([mod], "m", lazy.spawn("thunderbird"), desc="Launch Email"),
Key([mod], "g", lazy.spawn("gimp"), desc="Launch GIMP"),

# Custom window management
Key([mod, "shift"], "f", lazy.window.toggle_fullscreen(), desc="Toggle fullscreen"),
Key([mod], "bracketleft", lazy.screen.prev_group(), desc="Previous workspace"),
Key([mod], "bracketright", lazy.screen.next_group(), desc="Next workspace"),

# Volume control with custom increments
Key([], "XF86AudioRaiseVolume", lazy.spawn("amixer set Master 3%+"), desc="Volume up"),
Key([], "XF86AudioLowerVolume", lazy.spawn("amixer set Master 3%-"), desc="Volume down"),

# Screenshot shortcuts
Key([mod], "s", lazy.spawn("scrot -s"), desc="Screenshot selection"),
Key([mod, "shift"], "s", lazy.spawn("scrot"), desc="Screenshot full"),

Key Modifier Combinations

# Available modifiers:
mod = "mod4"        # Super/Windows key
"mod1"              # Alt key
"control"           # Ctrl key
"shift"             # Shift key

# Multiple modifier examples:
Key([mod, "shift"], "Return", lazy.spawn("alacritty --class floating"))
Key([mod, "control"], "space", lazy.widget["keyboardlayout"].next_keyboard())
Key(["mod1", "control"], "t", lazy.spawn("gnome-terminal"))
Testing Changes: After modifying your config, restart Qtile with Super + Ctrl + r to apply changes. Test new keybindings in a safe environment first.
