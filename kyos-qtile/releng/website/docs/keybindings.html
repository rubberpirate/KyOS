KyOS Keybindings Reference

Complete keyboard shortcuts and keybindings for efficient navigation and control of your KyOS desktop.

Essential Shortcuts

These are the most important keybindings you should memorize first:

Super + Enter Open Terminal

Opens the default terminal emulator (Alacritty)

Super + r Application Launcher

Opens dmenu for launching applications

Super + q Close Window

Closes the currently focused window

Super + Tab Next Window

Cycles through windows in current workspace

Super + [1-9] Switch Workspace

Switches to workspace 1-9

Super + Ctrl + r Restart Qtile

Restarts the window manager (applies config changes)

Mouse Actions

While KyOS is keyboard-focused, mouse actions are also supported:

🖱️ Window Operations

  • Click window: Focus window
  • Super + Left click: Drag floating window
  • Super + Right click: Resize floating window
  • Middle click titlebar: Close window

📋 Workspace Bar

  • Click workspace: Switch to workspace
  • Scroll on bar: Change workspace
  • Right click workspace: Context menu

🔊 System Tray

  • Click volume: Toggle mute
  • Scroll volume: Change volume
  • Click network: Network manager
  • Click clock: Calendar

Customizing Keybindings

You can customize keybindings by editing the Qtile configuration file:

# Edit Qtile configuration
nano ~/.config/qtile/config.py

# Example: Add custom keybinding
keys.extend([
    Key([mod], "b", lazy.spawn("firefox")),           # Super+b opens Firefox
    Key([mod, "shift"], "b", lazy.spawn("chromium")), # Super+Shift+b opens Chromium
    Key([mod], "c", lazy.spawn("code")),              # Super+c opens VS Code
    Key([mod], "t", lazy.spawn("thunar")),            # Super+t opens file manager
])

Key Modifiers

  • mod - Super/Windows key
  • "shift" - Shift key
  • "control" - Ctrl key
  • "mod1" - Alt key
  • "mod4" - Super key (alternative)

Special Keys

  • "Return" - Enter key
  • "space" - Spacebar
  • "Tab" - Tab key
  • "Escape" - Escape key
  • "BackSpace" - Backspace key
  • "Delete" - Delete key

Applying Changes

# After editing config.py, restart Qtile
Super + Ctrl + r

# Or from terminal
qtile cmd-obj -o cmd -f restart
