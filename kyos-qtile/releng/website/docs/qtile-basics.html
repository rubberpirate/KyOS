Qtile Window Manager Basics

Learn the fundamentals of Qtile, the powerful Python-based window manager that powers KyOS.

What is Qtile?

Qtile is a full-featured, hackable tiling window manager written and configured in Python. It's designed to be simple, small, and extensible. In KyOS, Qtile provides:

ü™ü Tiling Window Management

Windows automatically arrange themselves to efficiently use screen space without overlapping.

üêç Python Configuration

Configure everything using Python code, making it extremely flexible and programmable.

üè¢ Multiple Workspaces

Organize applications across different virtual desktops for better workflow management.

üìä Customizable Bar

Flexible status bar with widgets for system information, notifications, and more.

Window Layouts

Qtile supports multiple layout algorithms. KyOS comes pre-configured with the most useful ones:

MonadTall (Default)

Main

Switching Layouts

# Key combinations to switch layouts
Super + Space          # Cycle through available layouts
Super + Shift + Space  # Cycle backwards through layouts

# Or use specific layout shortcuts (if configured)
Super + m              # Switch to Max layout
Super + t              # Switch to MonadTall layout
Super + f              # Toggle floating for current window

Workspace Navigation

# Switch to workspace
Super + [1-9]              # Switch to workspace 1-9
Super + Left/Right         # Move between adjacent workspaces

# Move windows between workspaces
Super + Shift + [1-9]      # Move current window to workspace 1-9
Super + Shift + Left/Right # Move window to adjacent workspace

# Move window and follow it
Super + Ctrl + [1-9]       # Move window and switch to that workspace

Bar Widgets Explained

  • GroupBox: Shows all workspaces and highlights the active one
  • WindowName: Displays the title of the currently focused window
  • Systray: Shows system tray icons from running applications
  • Volume: Current audio volume level
  • Net: Network connection status and activity
  • Clock: Current date and time

Window Management

Focus and Selection

Qtile uses keyboard-driven window management:

# Navigate between windows
Super + h/j/k/l           # Move focus left/down/up/right
Super + Tab               # Cycle through windows
Alt + Tab                 # Traditional window switcher

# Window operations
Super + q                 # Close current window
Super + Shift + q         # Kill current window (force close)
Super + f                 # Toggle fullscreen
Super + Shift + f         # Toggle floating mode

Resizing Windows

# Resize windows (in tiling layouts)
Super + Ctrl + h/l        # Shrink/grow main window horizontally
Super + Ctrl + j/k        # Shrink/grow main window vertically

# Move windows around
Super + Shift + h/j/k/l   # Move window in that direction

# In floating mode
Super + Button1           # Drag window
Super + Button3           # Resize window

Window Rules

KyOS comes with smart window rules that automatically handle certain applications:

  • Dialog boxes: Automatically float
  • Games: Often fullscreen by default
  • Media players: May float for easier positioning
  • System tools: Configured for optimal workspace placement

Configuration Basics

Configuration File Location

Qtile configuration is stored in a Python file:

# Main configuration file
~/.config/qtile/config.py

# Backup the original before editing
cp ~/.config/qtile/config.py ~/.config/qtile/config.py.backup

# Edit with your favorite editor
nano ~/.config/qtile/config.py

Configuration Structure

The config file contains several main sections:

# Key bindings
keys = [
    Key([mod], "Return", lazy.spawn(terminal)),
    # ... more key bindings
]

# Window layouts
layouts = [
    layout.MonadTall(),
    layout.Max(),
    # ... more layouts
]

# Workspaces (groups)
groups = [Group(i) for i in "123456789"]

# Status bar configuration
screens = [
    Screen(
        top=bar.Bar([
            widget.GroupBox(),
            widget.WindowName(),
            # ... more widgets
        ], 30),
    ),
]

Applying Changes

After editing the configuration:

# Test configuration syntax
python ~/.config/qtile/config.py

# If no errors, restart Qtile to apply changes
Super + Ctrl + r

# Or restart from terminal
qtile cmd-obj -o cmd -f restart

Common Customizations

Adding New Key Bindings

# Example: Add key binding to open file manager
keys.extend([
    Key([mod], "e", lazy.spawn("thunar")),
    Key([mod], "w", lazy.spawn("firefox")),
    Key([mod, "shift"], "s", lazy.spawn("flameshot gui")),
])

Customizing the Bar

# Add/modify widgets in the bar
widget_list = [
    widget.GroupBox(
        active=colors[2],           # Color for active workspace
        inactive=colors[7],         # Color for inactive workspaces
        rounded=False,              # Square workspace indicators
        highlight_method="line",    # Underline active workspace
    ),
    widget.Sep(linewidth=0, padding=6),
    widget.WindowName(foreground=colors[6]),
    widget.Spacer(),
    widget.CPU(foreground=colors[4]),
    widget.Memory(foreground=colors[8]),
    widget.Clock(format="%Y-%m-%d %a %I:%M %p"),
]

Adding New Layouts

# Add more layouts to your config
layouts = [
    layout.MonadTall(margin=8),
    layout.MonadWide(margin=8),
    layout.Max(),
    layout.Floating(),
    layout.Columns(border_width=2),
    layout.TreeTab(),
]
