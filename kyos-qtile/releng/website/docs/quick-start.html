Quick Start Guide

Get up and running with KyOS in minutes. This guide covers installation, first boot, and essential tasks.

‚ö° 5-Minute Installation

1

Download & Create USB

  • Download KyOS ISO from the official website
  • Create bootable USB with Rufus (Windows) or dd (Linux/macOS)
  • Boot from USB (press F12/F2 during startup)
2

Run Installer

  • Open terminal and run: kyosinstall
  • Choose installation type (Full Disk recommended for beginners)
  • Follow the on-screen prompts
3

First Boot

  • Remove USB and reboot
  • Login with your created user account
  • Enjoy your new KyOS desktop!

üéØ Essential First Tasks

1. Update Your System

Always start by updating your system:

# Update package database and install updates
doas pacman -Syu

# This may take a few minutes depending on updates available

2. Connect to WiFi (if needed)

# List available networks
nmcli device wifi list

# Connect to your network
nmcli device wifi connect "YourNetworkName" password "YourPassword"

# Check connection
ping -c 3 google.com

3. Install Essential Software

# Web browser (Firefox is pre-installed, but you can add others)
doas pacman -S chromium

# Text editor with GUI
doas pacman -S gedit

# File manager
doas pacman -S thunar

# Media player
doas pacman -S vlc

# Office suite
doas pacman -S libreoffice-fresh

4. Set Up AUR Helper (for additional software)

# Install yay AUR helper
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si

# Now you can install AUR packages
yay -S visual-studio-code-bin
yay -S spotify
yay -S discord

üñ•Ô∏è Desktop Basics

Essential Keyboard Shortcuts

Super + Enter Open Terminal
Super + r Application Launcher
Super + q Close Window
Super + Tab Switch Windows
Super + [1-9] Switch Workspace
Super + Shift + [1-9] Move Window to Workspace
Super + f Fullscreen Window
Super + Shift + r Restart Qtile

Window Management

  • Tiling: Windows automatically tile to fill space
  • Floating: Press Super + t to toggle floating mode
  • Resizing: Use Super + h/j/k/l to resize windows
  • Moving: Super + Shift + h/j/k/l to move windows

Workspaces

KyOS comes with 9 workspaces by default:

  • Workspace 1: General applications
  • Workspace 2: Web browsing
  • Workspace 3: Development
  • Workspace 4: Files and documents
  • Workspaces 5-9: Available for your use

Installing Software

Official Repositories

# Search for software
pacman -Ss firefox

# Install software
doas pacman -S firefox

# Remove software
doas pacman -R firefox

# Search installed packages
pacman -Q | grep firefox

AUR (User Repository)

# Search AUR packages
yay -Ss spotify

# Install from AUR
yay -S spotify

# Update AUR packages
yay -Sua

File Management

# Open file manager
thunar

# Or use terminal commands
ls -la          # List files
cd ~/Documents  # Change directory
mkdir newfolder # Create directory
cp file1 file2  # Copy file
mv file1 file2  # Move/rename file
rm file1        # Delete file

System Information

# System information
neofetch

# Disk usage
df -h

# Memory usage
free -h

# Running processes
top

# Hardware information
lscpu
lspci
lsusb

üé® Customization Quick Start

Change Wallpaper

# Using nitrogen (pre-installed)
nitrogen

# Or using feh
feh --bg-scale /path/to/wallpaper.jpg

# Make it permanent by adding to autostart

Customize Terminal

Edit Alacritty terminal configuration:

# Edit terminal config
nano ~/.config/alacritty/alacritty.yml

# Change font, colors, transparency, etc.
# Restart terminal to see changes

Qtile Configuration

The main Qtile config is in Python:

# Edit Qtile configuration
nano ~/.config/qtile/config.py

# After making changes, restart Qtile
# Press Super + Ctrl + r

üöÄ Pro Tips

üí° Tip 1: Command Line Efficiency

Learn these commands to become more efficient:

  • history - View command history
  • Ctrl + r - Search command history
  • !! - Repeat last command
  • doas !! - Run last command with admin privileges

üîß Tip 2: Package Management

Useful pacman commands:

  • pacman -Qdt - Find orphaned packages
  • pacman -Qs package - Search installed packages
  • pacman -Qi package - Show package info
  • pactree package - Show package dependencies

‚ö° Tip 3: Performance

Optimize your system:

  • Enable SSD TRIM: doas systemctl enable fstrim.timer
  • Use fastest mirrors: doas reflector --save /etc/pacman.d/mirrorlist
  • Clean package cache: doas pacman -Sc

üõ°Ô∏è Tip 4: Security

Keep your system secure:

  • Update regularly: doas pacman -Syu
  • Enable firewall: doas ufw enable
  • Use strong passwords
  • Don't run unknown scripts as root
